# Iris Flower Classification

A simple Machine Learning project that classifies Iris flowers into three species using the Random Forest Classifier.

## Features
- Uses the built-in Iris dataset from Scikit-learn
- Trains a Random Forest model
- Evaluates model accuracy
- Generates a confusion matrix
- Predicts the species of a sample flower

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Installation

```bash
pip install pandas scikit-learn matplotlib seaborn
```

## Run the Project

```bash
python iris_classifier.py
```

## Output
- Displays dataset preview
- Prints model accuracy
- Shows classification report
- Saves `confusion_matrix.png`
- Predicts the species of a sample flower

## Dataset
The project uses the Iris dataset available in Scikit-learn.
